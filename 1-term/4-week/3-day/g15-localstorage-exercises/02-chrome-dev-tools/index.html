02-chrome-dev-tools

Step 1: Resources Tab

  1. Open your Chrome Developer Tools
  2. Navigate to the Resources Tab
  3. Expand Local Storage and find your domain
  4. View the keys and values you set in the last exercise
  5. Double click on the secret value and change it to 9876
  6. Type localStorage.getItem('secret') in your console and observe the value

You now know how to view all the localStorage values in Chrome Dev Tools.

You might notice a bunch of other types of storages available.

But, this exercise is only about localStorage

Congrats! Go to the next step!
