CSS Animation with Transform

What is a transform?

The Transform property let you modify elements in their coordinate space by skewing, rotating, translating, and scaling.

CSS Syntax

transform: function(parameters)

The 2D Transform Methods:

  • skew(deg)

    skews an element along the X and Y-axis by the given angle.

    transform: skewX(20deg);
  • scale(width, height)

    Increases or decreases the size of an element.

    Note: When scaling an element, it will still maintain its original box model, although appearing scaled.

    transform: scale(2,2);
  • rotate(deg) or rotate(turn)

    Rotates an element clockwise or counter-clockwise according to a given degree.

    transform: rotate(360deg);
  • translate(X-axis, Y-axis)

    Moves an element from its current position.

    transform: translateX(900px) rotate(1turn)

Transition-timing function

MDN states "The transition-timing-function CSS property is used to describe how the intermediate values of the CSS properties being affected by a transition effect are calculated. This in essence lets you establish an acceleration curve, so that the speed of the transition can vary over its duration."

    Keyword Values:
  • linear

    same speed from start to finish

  • ease (default)

    starts slow and ends slow

  • ease-in

    starts slow and ends fast

  • ease-out

    starts fast and ends slow

  • ease-in-out

    starts slow, fast in the middle and ends slow

  • cubic-bezier curve

    To Create Custom Timing Function: cubic-bezier curve
